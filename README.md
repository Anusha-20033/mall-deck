# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

## 🛠 Tech Stack

- React.js
- Vite
- Tailwind CSS
- GSAP (Scroll animations)
- Framer Motion
- JavaScript (ES6)

---

## 🎯 Project Overview

This project is a **Digideck-style interactive presentation**, not a traditional website.

It showcases Dubai Mall as a global luxury retail and entertainment destination through a cinematic storytelling experience.

Each section behaves like a **slide in a digital pitch deck**.

---

## ✨ Features

- Hero video landing section
- Scroll-based storytelling navigation
- Luxury parallax section (Framer Motion)
- Interactive Explore Experience gallery
- Events & retail showcase sections
- Smooth GSAP animations
- Mobile responsive design

---

## 🧠 Design Decisions

- Built as a **presentation deck, not a website**
- Each section represents a storytelling slide
- Focus on visual impact and minimal UI
- Used scroll animations to simulate guided presentation flow
- Image interactions added for engagement

---

## 🤖 AI Usage

AI tools were used for:

- Structuring component architecture
- Debugging Vite and React errors
- Improving UI layout decisions
- Suggesting animation patterns (GSAP + Framer Motion)

---

## 🚀 Improvements (Future Scope)

- Full-screen slide transitions (like PowerPoint cinematic mode)
- Scroll timeline navigation indicator
- Advanced animation sequencing
- Audio/ambient background integration
- More immersive 3D transitions

---

## 📌 Conclusion

This project demonstrates a **modern digital presentation system** designed for luxury brand storytelling and high-impact visual communication.
